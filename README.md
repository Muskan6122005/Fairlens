# 🔍 FairLens — AI Bias Detection Platform

> Your AI Is Biased. We Can Prove It.

FairLens is a premium, cyberpunk-themed AI bias detection platform that scans datasets and exposes hidden discrimination in 30 seconds.

**Live Demos:**
* Frontend: [https://fairlens-one.vercel.app/](https://fairlens-one.vercel.app/)
* Backend API: [https://fairlens-1rfu.onrender.com](https://fairlens-1rfu.onrender.com)

---

## ✨ Features

* **Instant Bias Scanning**: Upload any CSV dataset and run a disparate impact and demographic parity analysis across dimensions like Gender, Age, Location, Race, and Income.
* **Google Gemini Integration**: Generate plain-English, non-technical explanations of complex bias scores to help businesses understand their legal exposure.
* **Bias Fingerprint**: Visualize your dataset's unique bias profile through an interactive, multi-dimensional mandala.
* **Shadow Mode**: Input a specific demographic profile and see how your dataset's implicit biases would alter their outcomes.
* **Bias Battle Arena**: Upload two datasets and watch them go head-to-head in a visual showdown to see which model is more fair.

## 🚀 Future Expectations

* **Automated Fairness Correction Engine**: Moving beyond visualization, FairLens will seamlessly generate a mathematically re-weighted, fairness-optimized CSV that teams can download and immediately use for retraining.
* **Continuous Monitoring Dashboards**: Real-time APIs to plug FairLens into live production models, instantly flagging when data drift introduces new demographic disparities.
* **Expanded Compliance Reports**: Automatically generated PDF reports tailored perfectly to India's DPDP Act 2023, the EU AI Act, and major US Civil Rights legislation frameworks.

## 🛠️ Quick Start (Local Setup)

### Step 1: Install Python Dependencies

```bash
cd backend
pip install -r requirements.txt
```

### Step 2: Run the Backend

```bash
cd backend
python app.py
```

The Flask server will start at `http://localhost:5000`

### Step 3: Open the Frontend

Open `frontend/index.html` in your browser, or use a simple local server:

```bash
cd frontend
python -m http.server 8080
```

Then visit `http://localhost:8080`
